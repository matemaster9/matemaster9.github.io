二叉树

存储结构

利用数组的下标与完全二叉树的一一对应，满二叉树的数组表示。因为我们是按照满二叉树的编号来分配结点的存储空间，也就是无论满不满，都会给到2^h^-1的空间（实际数组长度为2^h^）

这样对于第i个结点存在以下结论：

  • 左孩子：2i
  • 右孩子：2i + 1
  • 父结点：floor(i/2)

线性存储结构：数组

public class BiTree<E> {
    
    private TreeElem[] tree;
    
    private class TreeElem {
        E elem;
        boolean isEmpty; // 代表当前结点元素是否存在！
    }
}

链式存储结构：类似链表

对于链表的存储结构，我们发现对于一颗二叉树来说，假设它的结点个数为n，那么它最多存在2n个链接域（也就是左右链接），但是这2n个域，只有n-1个域被使用，因为每一个结点只会和父结点直接存在链接，那么就会有n+1个域是闲置的（这里闲置的域在二叉树的线索化存在用处）

public class BiTree<E> {
    
    private TreeNode root;
    
    private class TreeNode {
        E elem;
        TreeNode left,right;
    }
}

递归遍历

从左至右分别是：DLR先序遍历(preOrder)、LDR中序遍历(inOrder)、LDR后续遍历(postOrder)

先序遍历DLR

public void preOrder() {
    if(isEmpty()) return;
    preOrder(root);
}

private void preOrder(TreeNode x) {
    if(x == null) return;
    System.out.print(x.key);
    if(x.left != null) preOrder(x.left);
    if(x.right != null) preOrder(x.right);
}

中序遍历LDR

public void inOrder() {
    if(isEmpty()) return;
    inOrder(root);
}

private void inOrder(TreeNode x) {
    if(x == null) return;
    if(x.left != null) preOrder(x.left);
    System.out.print(x.key);
    if(x.right != null) preOrder(x.right);
}

后序遍历LRD

public void postOrder() {
    if(isEmpty()) return;
    postOrder(root);
}

private void postOrder(TreeNode x) {
    if(x == null) return;
    if(x.left != null) preOrder(x.left);
    if(x.right != null) preOrder(x.right);
    System.out.print(x.key);   
}

三种递归的迭代实现

public void preOrder() {
    TreeNode x = root;
    // 利用辅助栈，保存结点的双亲结点信息
    Stack<TreeNode> parent = new Stack<>();
    parent.push(null);
    while(x != null) {
        System.out.print(x.key);
        if(x.left != null) {
            parent.push(x);
            x = x.left;
        }    
        else if(x.right != null) {
            parent.push(x);
            x =x.right;
        }    
        else   x = parent.pop();                  
    }
}

public void inOrder() {
    TreeNode x = root;
    Stack<TreeNode> parent = new Stack<>();
    while(x != null) {
        if(x.left != null) {
            parent.push(x);
            x = x.left;
        } else {
            x = parent.pop();
        }
        System.out.print(x.key);
        if(x.right != null) {
            parent.push(x)
            x =x.right;
        } else {
            x = parent.pop();
        }
    }
}

public void postOrder() {
    TreeNode x = root;
    Stack<TreeNode> parent = new Stack<>();
    while(x != null) {
        if(x.left != null) {
            parent.push(x);
            x = x.left;
        } else {
            x = parent.pop();
        }
        if(x.right != null) {
            parent.push(x)
            x =x.right;
        } else {
            x = parent.pop();
        }
        System.out.print(x.key);
    }
}

层次遍历

结合队列结构实现二叉树的层次遍历，这里是基于链式存储结构的二叉树编写的层次遍历。通过辅助队列queue，每一个根结点的出队，都需要检查其子结点是否存在，存在就入队！，知道所有队列元素全部出队，结束！

层次遍历算法思想：

  • 初始化辅助队列
  • 二叉树根结点入队
  • 辅助队列非空？yes就出队，并且返回出队元素的子结点，如存在就入队
  • 重复第三步，知道队列为空
public Iterable<E> levelOrder() {
    Queue<E> elems = new LinkedList<>();
    Queue<TreeNode> queue = new LinkedList<>();
    queue.add(root);
    while (!queue.isEmpty()) {
        Node x = queue.poll();
        if (x == null) continue;
        elems.add(x.key);
        if(x.left != null) {
            queue.add(x.left);
        }
        if(x.right != null) {
            queue.add(x.right);
        }
    }
    return elems;
}

遍历确定结构

通过上面的遍历方式，以及之前查找算法中，不同的二叉树结构可以表示相同的有序键列可以知道，单独的遍历方式，是无法确定二叉树的具体结构，但是通过不同遍历的组合，我们就可以确定二叉树结构。

  • 前序+中序
  • 后序+中序
  • 层次+中序

线索二叉树

中序线索化

我们在查找算法中指出：中序遍历得到的序列是符合二叉排序树的顺序的

之前在采用链式存储指出了闲置的链接域，这里的作用就是：让闲置域指向结点的直接前驱和后继结点。为了区别实线和虚线链接的区别，我们在链式存储结构上定义标志位 ltag、rtag。当tag=0时，表示实线链接，链接是指向左右孩子的，当tag=1时，表示虚线链接，链接是指向前驱和后继结点的。

线索化的二叉树的优势在于：方便寻找结点的前驱和后继结点。

public class ThreadBiTree<E> {
    
    private ThreadNode root;
    
    private class ThreadNode {
        E data;
        ThreadNode left, right;
        int ltag, rtag;
    }
}

如何实现对二叉树中序线索化？

首先需要理解的是，被线索化的结点都是0<=degree<2的结点。我们利用链接prev保存当前结点链接curr的前驱，curr保存当前结点链接。

算法思想：

  • 判断当前链接的左链接是否为空，如果是空的话，将左链接指向当前结点的前驱结点，并将标志位ltag=1。
  • 判断前驱结点prev的右链接是否为空，如果是空的话，将右链接指向前驱prev的后继结点，将标志位rtag=1
  • 将当前结点链接指向中序遍历的下一个结点
  • 重复上面三步，直到所有度不足2的结点全部线索化
public void inThread() {
    Node prev = null;
    threadForInOrder(root, prev);
    // 对于最右结点，它的标志位因为递归结束，无法进行修改，所以应该在递归调用后修改
    if(prev.right == null) {
        prev.rtag = 1;
    }
}

private void threadForInOrder(ThreadNode curr, ThreadNode prev) {
    if(curr != null) {
        threadForInOrder(curr.left, prev);
        //---------对当前结点的操作-------------
        if(curr.left == null) {
            curr.left = prev;
            curr.ltag = 1;
        }
        if(prev != null&& prev.right == null) {
            prev.right = curr;
            prev.rtag = 1;
        }
        prev = curr;
        //-----------------------------------
        threadForInOrder(curr.right, prev);
    }
}

线索化的目的是：让我们在遍历时，可以方便查找每个结点的前驱和后继。那么对于一个已经线索化的二叉树如何找寻结点的前驱和后继？

对于一个二叉树的根结点来说：当存在子树的情况下，它的前驱结点就是：左子树上的最右结点。后继结点就是：右子树上的最左结点！当不存在子树的情况下，利用已经线索化的结点，就可以获取对应的前驱和后继！

找寻二叉树的最左结点

public ThreadNode leftmost() {
    if(root == null) throw new IllegalArgumentException("tree is null");
    return leftmost(root);
}

private ThreadNode leftmost(ThreadNode x) {
    // assert x != null
    if(x.left == null) return x;
    else               return leftmost(x.left);
}

找寻二叉树的最右结点

public ThreadNode rightmost() { 
    if(root == null) throw new IllegalArgumentException("tree is null");
    return rightmost(root); 
} 
private ThreadNode rightmost(ThreadNode x) { 
    if(x.right == null) return x; 
    else 				return rightmost(x.right); 
}

找寻二叉树的任一结点x的前驱和后继

public ThreadNode getPrev(ThreadNode x) {
    if(x.ltag == 1) return x.left;
    else            return rightmost(x.left);
}

public ThreadNode getSucc(ThreadNode x) {
    if(x.ltag == 1) return x.right;
    else            return leftmost(x.right);
}

平衡二叉树

首先必须搞明白的问题是：什么是平衡二叉树？如何让二叉树平衡？

AVL：代表树中的任意一个结点，它的左右子树高度只差不会大于1。（也就是说，每一个结点都是平衡二叉树）
我们假设定义结点的平衡因子balance = left - right。也就是|balacne| <= 1。我就以ALLOWED_IMBALANCE = 1为界限讨论四种情况！

由此我们可以得到平衡二叉树的结构定义：

public class AvlTree<E> {
    
    // 平衡二叉树的根结点和平衡因子
    private AvlNode avlRoot;
    private static final int ALLOWED_IMBALANCE = 1;
    
    // AVL树结点
    private class AvlNode {
        E elem;
        AvlNode left, right;
        int height;

        public AvlNode(E elem) {
            this(elem,null,null,0);
        }

        private AvlNode(E elem, AvlNode left, AvlNode right, int height) {
            this.elem = elem;
            this.left = left;
            this.right = right;
            this.height = height;
        }
    }
    
    private int height(AvlNode x) {
        return x == null ? -1 : x.height;
    }
    
    // 此棵平衡二叉树的高度
    public int height() {
        return height(avlRoot);
    }
}

知晓了平衡二叉排序树的概念，我们如何保证该二叉树是平衡的？

首先需要知道让二叉树失去平衡的四种情况：

  • 向结点的左孩子的左子树插入一个结点（外部）
  • 向结点的右孩子的右子树插入一个结点（外部）
  • 向结点的左孩子的右子树插入一个结点（内部）
  • 向结点的右孩子的左子树插入一个结点（内部）

之所以分为内外部，因为在外部插入结点导致的失衡那么通过单次旋转即可实现，而在内部插入结点导致的失衡那么需要两次单旋转才可实现。废话太多，上图解

（1）单旋

（2）双旋

通过图解我们可以轻松看出，应对四种情况二叉树的旋转操作。那么我们可以通过代码实现上述的四种操作！

// 外部，左孩子的左子树
private AvlNode rotateForLL(AvlNode parent) {
    if (parent == null) throw new IllegalArgumentException("node is null");
    AvlNode child = parent.left;
    parent.left = child.right;
    child.right = parent;
    parent.height = Math.max(height(parent.left),height(parent.right)) + 1;
    child.height = Math.max(height(child.left), parent.height) + 1;
    return child;
}

// 外部，右孩子的右子树
private AvlNode rotateForRR(AvlNode parent) {
    if (parent == null) throw new IllegalArgumentException("node is null");
    AvlNode child = parent.right;
    parent.right = child.left;
    child.left = parent;
    parent.height = Math.max(height(parent.left), height(parent.right)) + 1;
    child.height = Math.max(parent.height, height(child.right)) + 1;
    return child;
}

// 内部，左孩子的右子树
private AvlNode rotateForLR(AvlNode grandpa) {
    AvlNode parent = grandpa.left;
    AvlNode child = rotateForRR(parent);
    return rotateForLL(child);
}

// 内部，右孩子的左子树
private AvlNode rotateForRL(AvlNode grandpa) {
    AvlNode parent = grandpa.right;
    AvlNode child = rotateForLL(parent);
    return rotateForRR(child);
}

知晓如何恢复二叉树的平衡结构，接下来我们就应该利用恢复手段，生成平衡二叉排序数，以此解决生成过程中的不平衡现象！

首先需要做的是：确定不平衡的情况是上述四种的哪一种！
其次就是：根据不同的情况所需要采取的不同的平衡手段！

根据不同的情况采取的不同的平衡手段

// 四种情况的应对措施，确保高度一致时，采用单旋转
private AvlNode balance(AvlNode x) {
    if (x == null) return null;
    if (height(x.left) - height(x.right) > ALLOWED_IMBALANCE) {
        if (height(x.left.left) >= height(x.left.right)) {
            x = rotateForLL(x);
        } else {
            x = rotateForLR(x);
        }
    } else if (height(x.right) - height(x.left) > ALLOWED_IMBALANCE) {
        if (height(x.right.right) >= height(x.right.left)) {
            x = rotateForRR(x);
        } else {
            x = rotateForRL(x);
        }
    }
    x.height = Math.max(height(x.left),height(x.right)) + 1;
    return x;
}

private AvlNode insert(AvlNode x, E elem) {
    if(x == null) {
        return new AvlNode(elem);
    }
    int cmp = elem.compareTo(x.elem);
    if (cmp < 0) {
        x.left = insert(x.left,elem);
    } else if (cmp > 0) {
        x.right = insert(x.right, elem);
    }
    return balance(x);
}

public void insert(E elem) {

private AvlNode delete(AvlNode x, E elem) {
    if (x == null) return null;
    int cmp = elem.compareTo(x.elem);
    if (cmp < 0) {
        x.left = delete(x.left, elem);

public List<E> elemList() {
    List<E> list = new ArrayList<>();
    if (avlRoot == null) return list;
    else                 return inOrderList(avlRoot, list);
}

private List<E> inOrderList(AvlNode x, List<E> list) {
    // assert x != null
    if (x.left != null) {
        inOrderList(x.left, list);
    }
    list.add(x.elem);
    if (x.right != null) {
        inOrderList(x.right,list);
    }
    return list;
}
